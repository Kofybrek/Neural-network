## 运行

- 通过`Makefile`文件运行

  - `make`
  - `make run`

## C++源码

- `FRAME_DURATION` 和 `lag`是控制帧率的参数

  - `FRAME_DURATION`: 每帧的时间间隔, 单位是毫秒

    60 帧
